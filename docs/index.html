Liquidis

A browser-based 2D canvas framework

Examples

After cloning this repository, navigate to the examples directory and run the following commands:

npm install
gulp production

Navigate to the examples directory in your browser and you should see different shapes being rendered.

Documentation

The documentation in the docs directory is generated by jsDoc.

JSDoc CSS Styles

Make sure to move site.codeboxsystems.css into liquidis/node_modules/gulp-jsdoc/node_modules/ink-docstrap/template/static/styles/site.codeboxsystems.css

Testing

Units tests are locations in the tests directory and run using Karma.
